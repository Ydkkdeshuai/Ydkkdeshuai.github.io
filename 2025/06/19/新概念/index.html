3、运放

3.1理想运放(56-57)

理想运放特点：

$A_{uo} →∞ ，f_H→∞，f_L→0$

输入阻抗$R_i→∞$

输出阻抗$R_o→0$

$u_o = A_{uo}(u_+-u_-)$

虚短：引入负反馈，运放的放大倍数$A_{uo} →∞，u+=u-$。

虚断：输入阻抗无穷大，电流近似为零

3.2 负反馈理论(58-63)

判断正负反馈：

环路极性法：施加小信号从回路判断

方框图分析法：

传递函数：$\frac{u_o(s)}{u_i(s)} = M\frac{A_{uo}}{1+FA_{uo}} =M \frac{G(s)}{1+FG(s)} $

负反馈对放大性能影响：

增益稳定性：开环增益变化很大，但闭环增益变化很小。

上线截止频率提高：

下限截止频率降低：

输入电阻：

​ 串联负反馈增大输入电阻

​ 并联负反馈减小输入电阻

输出电阻：

​ 一般会降低输出电阻

要会求输入输出电阻，不同放大电路的输入输出电阻不同，需要考虑。如差分放大电路、同相比例放大电路(并联负反馈)、反向放大比例电路(串联负反馈)。

负反馈对失真度的影响

原因：实际输入输出关系不为纯线性，存在非线性。

实际可能表达式：$u_o = A_{uo}(u_{id} + k_2u_{id}^2 + k_3u_{id}^3 … … + k_nu_{id}^n)$

本质：负反馈稳定了对于不同幅值信号的闭环增益，对于不同幅值信号有效降低失真度。

3.3 负反馈放大电路的分析方法(64-67)

虚短虚断法：

优点：简单快捷

缺点：

当输入信号频率上升，运放对于该频率的放大倍数不够大会失效。即不满足 $1<<|FG(s)|$。

看不到相位关系

大运放法：多个运放电路用一个代替，简化分析。

环路方程法：求传递函数。

3.4 实际运放(68-73)

理想运放 实际运放
$A_{uo}$ ∞ 有限、随频率下降(GBW)
$R_i$ ∞ 较大
$R_o$ 0 较小
线性关系 线性 小范围幅值，小范围频率近似线性

自制运放：

要求：

正负电源供电、放大倍数足够高、输入阻抗大、输出阻抗小

组成：

差分输入级

中间放大级

输出级。

电流源：静态工作点 负载

运放关键参数(70)：

输入失调电压 VOS: Input Offset Voltage：运放内部电路的电路结构以及非对称性导致。

​ 输入失调电压对放大电路的影响：输入为零 输出不为零

$u_{out} = V_{os_out} -\frac{R_2}{R_1}u_{in} = V_{os}\frac{R_2+R_1}{R_2}-\frac{R_2}{R_1}u_{in}$

手册典型值和最大值含义：最大不会超过7，有68.27的概率在±3内。

输入偏置电流 IB:Input Bias Current：

运放工作需要静态工作电流$I_{BQ}：I_B = \frac{IB^+ + IB^−}{2}$

输入失调电流 IOS:Input Offset Current

内部电路的非对称性导致

$I_{OS} = |IB^+ − IB^−|$

输入偏置电流与输入失调电流近似相等,或者维持在一个数量级。

对输出失调电压的影响：

$U_{os_out} = \frac{RG + RF}{RG} (V_{OS} + I_B^− × R_G‖R_F − I_B^+ × R_{MASCH}) \=\frac{R_G + R_F}{R_G} (V_{OS} + I_B(R_G‖R_F − R_{MASCH}) − 0.5I_{OS}(R_G‖R_F + R_{MASCH}))$

所以

$令R_{MASCH} = R_G‖R_F能消除偏置电流对输出失调电流的影响。$

等效输入失调电压：多种原因导致的输出失调电压除以总增益，占测量信号的百分比就能衡量误差。

输入、输出电压范围:Input Voltage Range、Swing from rail：

轨到轨运放：输入电压轨接近供电轨(输入轨到轨)、输出电压轨接近供电轨(输出轨到轨)

共模抑制比:Common-mode rejection ratio,CMRR

$CMRR = \frac{A_d}{ A_c} = 20 log ( \frac{A_d}{ A_c}) (dB)$

增益带宽积：GBW

GBW：开环放大倍数等于直流放大倍数的 $\sqrt{2}$倍的频率。

GBW反应闭环放大电路的带宽能力。

UGBW：开环放大倍数等于1的频率。

开环电压增益:Open-loop gain,AVO 或者教科书常用的 Auo：

对于直流信号的放大倍数，随频率下降。

压摆率:Slew rate,SR

运放噪声参数(71)

输出端噪声原因：

运放本身的噪声呈现在输出端;

外围电路中的电阻噪声;

运放输入端电流噪声在外电路电阻上产生的电压噪声;

源自供电电源噪声的输出端噪声;

源自空间电磁干扰的输出端噪声;

源自周边系统的地线环路噪声;

具体没看

设计考虑因素：

供电电压

电阻大：电阻热噪声增加。
